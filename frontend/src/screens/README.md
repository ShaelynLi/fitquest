# Screens

This directory contains the main screen components of the application.

## Structure
```
screens/
├── auth/            # Authentication screens (Login, Register, etc.)
├── home/            # Home screen and related screens
├── profile/         # User profile screens
├── settings/        # Settings and configuration screens
└── README.md        # This file
```

## Usage
```javascript
// Import screens
import LoginScreen from '../screens/auth/LoginScreen';
import HomeScreen from '../screens/home/HomeScreen';
```
